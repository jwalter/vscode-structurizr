# structurizr README
This is my attempt at writing a Visual Studio Code extension for the Structurizr DSL (see References).

## Feature ideas
 - Syntax highlighting
 - Code completion
 - Use Structurizr CLI to convert DSL file to PlantUML/Mermaid/etc
 - Publish to structurizr.com using CLI
